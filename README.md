# Bioinformatics Course Assignments

This repository contains assignments and related materials for the undergraduate course **Bioinformatics**, offered in the **Spring Semester of the 2025 academic year at Tsinghua University**.

## Course Overview

Bioinformatics is an interdisciplinary field that analyzes biomedical data using mathematics, statistics, and computer science. This course aims to provide foundational knowledge in bioinformatics while cultivating practical analytical and programming skills.

Through progressive training, students learn:

- Basic Linux operations  
- Programming skills required for bioinformatics  
- Fundamental analysis methods for next-generation high-throughput sequencing data  
- Practical data mining and analytical reasoning skills  

The course emphasizes hands-on practice and computational problem-solving, strengthening the ability to explore and interpret complex biological data.
