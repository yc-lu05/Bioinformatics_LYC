## 二、生物信息学四步框架

1. **Question**：基于生物/医学知识提出问题  
2. **Information**：获取数据（DNA/RNA/Omics）  
3. **Analysis**：数据清洗与特征提取  
4. **Modeling**：建立模型 + 设计算法  



## 三、Model 与 Algorithm 的区别

### 1. Model（模型）

Model 是对数据规律的数学刻画，用于描述变量之间的关系。

在生物信息学中，模型用于表达：

- 基因表达与表型的关系  
- 序列的概率结构  
- 生物现象的统计依赖  

常见模型包括：

- 线性回归  
- 逻辑回归  
- 神经网络  

模型关注的是：数据之间“是什么关系”。

### 2. Algorithm（算法）

Algorithm 是为完成计算任务而设计的步骤与规则。

在生物信息学中，算法用于：

- 序列比对（动态规划）  
- 排序与搜索  
- 模型参数求解（如梯度下降）  

算法强调：

- 明确流程  
- 可执行性  
- 计算效率  

它解决的是：如何完成计算。

### 3. 二者关系

模型提供结构，算法实现求解。

例如：

- Logistic Regression 是模型  
- Gradient Descent 是求解该模型的算法  

模型决定我们如何理解数据，算法决定我们如何得到结果。
